### 数据分析要求
#### 3.1 数据摘要和可视化
**数据摘要**

 标称属性，给出每个可能取值的频数  
 
 数值属性，给出5数概括及缺失值的个数

**数据可视化**

 使用直方图、盒图等检查数据分布及离群点

#### 3.2 数据缺失的处理
观察数据集中缺失数据，分析其缺失的原因。分别使用下列四种策略对缺失值进行处理:

将缺失部分剔除

用最高频率值来填补缺失值

通过属性的相关关系来填补缺失值

通过数据对象之间的相似性来填补缺失值

注意：在处理后完成，要对比新旧数据集的差异。
